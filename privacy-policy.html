Политика конфиденциальности Local AI Assistant

Дата вступления в силу: 24 марта 2026 г.

1. Общая информация

Local AI Assistant — расширение Chrome для локального взаимодействия с AI в боковой панели браузера. Расширение разработано ASTUS LAB.

2. Какие данные обрабатываются

Расширение может обрабатывать:

  • текст, который пользователь вводит в чат вручную;
  • выделенный пользователем текст страницы (через контекстное меню «Спросить AI»);
  • текст текущей страницы только по явному действию пользователя (кнопка «Суммаризировать»).

3. Как используются данные

Данные используются исключительно для основных функций расширения:

  • генерация ответа ассистента;
  • суммаризация содержимого текущей страницы;
  • перенос выделенного текста в чат.

4. Передача и хранение данных

  • Расширение не продает пользовательские данные.
  • Расширение не передает пользовательские данные третьим лицам в рекламных целях.
  • Расширение не использует пользовательские данные для оценки кредитоспособности или кредитования.
  • Расширение не использует удаленный исполняемый JavaScript/Wasm-код.
  • Данные не отправляются на сервер разработчика как часть основной работы расширения.

5. Разрешения

Расширение запрашивает только разрешения, необходимые для работы заявленных функций:

  • sidePanel — показ интерфейса ассистента в боковой панели;
  • contextMenus — запуск ассистента по выделенному тексту;
  • storage — хранение служебного состояния;
  • activeTab и scripting — извлечение текста текущей страницы по запросу пользователя;
  • доступ к хостам (<all_urls>) — работа функций на разных сайтах по явному действию пользователя.

6. Безопасность

Мы принимаем разумные меры для ограничения обработки данных рамками заявленного функционала расширения.
