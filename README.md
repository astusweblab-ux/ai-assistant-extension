# AI Assistant — Chrome Extension (Prompt API / Gemini Nano)

Локальный AI-ассистент в боковой панели Chrome на базе встроенного Prompt API (Gemini Nano).  
Без API-ключей и без внешнего бэкенда.

---

## Быстрая установка

1. Запустите `SETUP.bat` (он покажет актуальные шаги для флагов)
2. Откройте `chrome://extensions/`
3. Включите **Режим разработчика**
4. Нажмите **Загрузить распакованное** и выберите папку `ai-assistant-extension`

---

## Требования Prompt API

Проверьте флаги в `chrome://flags`:

- `#optimization-guide-on-device-model`
- `#prompt-api-for-gemini-nano-multimodal-input`
- Если в вашей версии Chrome старые имена флагов, также включите: `#prompt-api-for-gemini-nano`

После изменения флагов перезапустите Chrome.

---

## Первый запуск

1. Откройте боковую панель расширения
2. Расширение проверит доступность Prompt API
3. Если модель не скачана, Chrome загрузит Gemini Nano автоматически
4. Если Chrome попросит пользовательский жест, нажмите кнопку **"Запустить модель"** в оверлее
5. После этого статус сменится на "Модель готова!"

Модель кэшируется: следующие запуски обычно быстрее.
Chrome Prompt API сейчас требует явный язык вывода (`en`, `es` или `ja`).
Расширение выбирает его автоматически по языку браузера (иначе использует `en`).

---

## Если видите ошибку "Prompt API не найден"

1. Обновите Chrome до актуальной версии
2. Включите флаги из раздела выше
3. Перезапустите Chrome
4. Проверьте `chrome://on-device-internals` (состояние on-device модели)
